- 数组不变，区间查询：前缀和、树状数组、线段树；
- 数组单点修改，区间查询：树状数组、线段树；
- 数组区间修改，单点查询：差分、线段树；
- 数组区间修改，区间查询：线段树。
- 看到「最大化最小值」或者「最小化最大值」就要想到二分答案，这是一个固定的套路。
- 为什么？一般来说，二分的值越大，越能/不能满足要求；二分的值越小，越不能/能满足要求，有单调性，可以二分。
